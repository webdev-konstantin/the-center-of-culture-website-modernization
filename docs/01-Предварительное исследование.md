# День 1 — Первичный reverse engineering и исследование системы

Дата: 2026-06-30

## Цель дня

Начать исследование существующей самописной CMS сайта ruscenter.hu и подготовить её к дальнейшей модернизации с использованием современного лендинга, экспортированного из Tilda.

## Выполненные задачи

### 1. Исследование архитектуры проекта

Проведён первоначальный анализ структуры существующего сайта.

Изучены:

- файловая структура проекта;
- расположение шаблонов;
- административная часть CMS;
- структура хранения страниц;
- организация контента.

### 2. Анализ базы данных MySQL

Исследована база данных проекта через phpMyAdmin.

В ходе анализа:

- определены основные таблицы CMS;
- найден механизм хранения страниц;
- исследованы связи между разделами;
- выявлены таблицы, отвечающие за виджеты.

### 3. Reverse Engineering CMS

Начато восстановление логики работы самописной CMS без исходной документации.

Выполнено:

- анализ поведения административной панели;
- исследование механизма формирования страниц;
- определение принципов хранения компонентов сайта.

### 4. Исследование виджета «Панель ярлыков»

Это оказалось наиболее трудоёмкой частью первого дня.

Удалось установить:

- где реально находятся значения виджета;
- каким образом CMS связывает записи базы данных с элементами страницы;
- почему виджет долгое время не отображался корректно.

В результате стало понятно, что значения находятся не внутри шаблонов, а хранятся непосредственно в базе данных MySQL.

### 5. Практическая проверка гипотез

После внесения найденных значений:

- восстановился раздел «Библиотека»;
- появились баннеры подразделов;
- CMS начала корректно отображать содержимое панели ярлыков.

Это подтвердило правильность выбранного направления исследования.
