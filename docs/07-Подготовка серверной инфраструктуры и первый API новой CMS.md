# Цель дня

Подготовить серверную инфраструктуру для дальнейшей интеграции новой главной страницы с существующей CMS без нарушения работы действующего сайта.

Необходимо было:

- создать полноценную точку восстановления;
- исследовать структуру базы данных;
- подготовить сервер к работе с API;
- развернуть тестовую версию новой главной страницы;
- проверить выполнение собственного PHP-кода на реальном сервере.

Планируемая архитектура:

```text
Новая главная
        ↓
PHP API
        ↓
JSON
        ↓
JavaScript
        ↓
Будущая интеграция с CMS
```
