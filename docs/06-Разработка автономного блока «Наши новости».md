# Цель дня

Полностью разработать автономный компонент **«Наши новости»**, который до запуска в онлайн должен работать автономно: без PHP, MySQL и существующей CMS.

Необходимо было создать независимый HTML/CSS/JavaScript-блок, максимально приближённый к будущему компоненту, который позднее будет получать данные из существующей CMS.

Планируемая архитектура:

```text
index.html
        ↓
HTML-разметка
        ↓
CSS
        ↓
JavaScript
        ↓
Готовый автономный компонент
        ↓
Будущая интеграция с CMS
```

## 7. Настройка блока «Студии»

Исследован файл:

```text
studios.css
```

Обнаружены параметры:

```css
.studios {
    padding: 85px 20px 90px;
}

.studios-header {
    margin-bottom: 110px;
}
```

Выполнена оптимизация.

Изменено:

```text
margin-bottom

110 px
        ↓
60 px
```

Интервал между заголовком и карточками стал визуально более компактным.

---

## 8. Настройка расстояния между секциями

Определено, что расстояние складывается из двух величин:

```text
.studios
padding-bottom

+

.content-carousel-section
padding-top
```

Выполнено уменьшение нижнего отступа секции:

```text
90 px
        ↓
40 px
```

После изменения блоки стали восприниматься как единая композиция.
